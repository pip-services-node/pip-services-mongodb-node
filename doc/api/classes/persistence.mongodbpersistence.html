Class MongoDbPersistence

Class that provides methods for working with MongoDB servers.

MongoDbPersistences can be configured using the configure method, which searches for and sets:

  • the connection resolver's connections and credentials ("connection(s)" and "credential(s)" sections);
  • the MongoDB collection to work with ("collection" parameter);
  • this persistence's options ("options" section):
    • "max_pool_size" (default is 2);
    • "keep_alive" (default is 1);
    • "connect_timeout" (default is 5000);
    • "auto_reconnect" (default is true);
    • "max_page_size" (default is 100);
    • "debug" (default is false).

A logger and a connection resolver can be referenced by passing the corresponding "logger", "discovery" (for the connection resolver), and "credential-store" (for the connection resolver's credential resolver) references to the object's setReferences method.
